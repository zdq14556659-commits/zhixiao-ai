智销AI 是一个面向全屋定制软件销售团队的 AI CRM，包含 Web 管理端和微信小程序。Web 与小程序共用同一套后端、账号权限和经营统计口径。

## 经营看板

- 按登录账号权限统计本人、单位、战区或全公司的真实经营数据。
- 支持月份、自定义日期和管理范围筛选，展示实际进款、签单、成交、名单、商机、漏斗、趋势、销售排名和单位排名。
- 商机以“完成有效演示”为准，成交必须填写合同金额；实际进款按进款日期单独统计，旧成交客户不会自动推算财务数据。
- 月度目标支持公司、战区、单位、销售四级，主管以上只能维护自身管理范围内目标。
- 小智经营分析只引用后端统计结果，不生成系统中不存在的业绩数字。

## 已实现模块

- 经营驾驶舱：真实进款与签单、阶段漏斗、90 天转化率、目标完成率、行动中心、行业画像和经营建议。
- 客户跟进转化：名单、线索、商机、成交四阶段看板。
- 跟进记录：销售可填写记录和下次跟进日期，支持逾期提醒。
- 主管日报：主管及以上角色可按日期筛选销售录入名单、新增线索、新增商机和跟进数量，并导出 CSV。
- App 拨号流程：手机号使用 `tel:` 直接拨号，录音分析流程已做成可交互演示。
- AI 通话分析：识别客户意向、异议、成交策略和下一步动作。
- 转化漏斗：自动计算阶段数量，识别薄弱环节并给出改进计划。
- 系统管理：员工开账号、角色权限、运营导入官方资源给主管，主管分配销售。
- 地推拜访：模拟全国地图、定位打卡、上传工厂名头、产线设备和现用软件。
- AI 话术助手：从知识库检索最佳应答，支持运营录入和成功案例扩充。

## 账号与密码安全

- 后端数据版本为 `backend-v5`，旧账号首次读取时会将明文密码自动迁移为 `scrypt` 加盐哈希，原密码仍可登录。
- Web 右上角和小程序「我的 - 账号安全」支持员工凭原密码修改密码；系统设置支持管理员在权限范围内重置员工密码。
- 修改或重置后会递增账号认证版本，所有设备上的旧令牌立即失效。新建、迁移和重置账号会持续提醒员工修改初始密码。
- 连续 5 次密码错误后，账号或来源 IP 暂停登录 10 分钟；安全日志只记录操作人、目标账号、类型、时间和来源 IP。
- 正式环境必须配置 `AUTH_TOKEN_SECRET`，建议使用至少 32 字节的随机字符串，并通过 HTTPS 对外提供服务。可在 PowerShell 运行：`node -e "console.log(require('crypto').randomBytes(48).toString('base64url'))"`。
- 密码安全回归测试：`npm run test:password`。

## 后续接入建议

- 小程序 MVP 已放在 `miniprogram/` 目录。用微信开发者工具导入该目录即可预览；当前 AppID 使用 `touristappid`，正式发布时替换成你自己的小程序 AppID。
- 小程序地推模块已支持图片上传、定位打卡、绿色/红色地图点和城市分布。高德 WebService 现在由后端代理调用，小程序不直接暴露 Key。
- 今天下午给员工地推使用，建议先走微信「体验版」：在微信开发者工具导入 `miniprogram/`，填你的小程序 AppID，上传版本，然后在微信公众平台把员工微信号加入体验成员。正式上线需要提交微信审核，通常不能保证当天通过。
- 当前小程序是本地缓存 MVP，适合今天演示和单机试用；如果要老板后台实时看到所有员工打卡，需要接后端或微信云开发数据库。
- 共享后端已放在 `backend/`，Web 端和小程序端都可以读写同一份数据。启动方式：在项目根目录运行 `powershell -ExecutionPolicy Bypass -File .\start-backend.ps1`，然后打开 `http://localhost:8787/index.html`。
- 小程序当前开发环境 API 地址在 `miniprogram/app.js` 的 `API_BASE`，默认 `http://127.0.0.1:8787/api`。给员工体验版/正式版使用时，必须把后端部署到 HTTPS 域名，并替换成 `https://你的域名/api`，同时在微信小程序后台配置该域名为 request/uploadFile 合法域名。
- 高德 WebService Key 已接入后端，后端通过 `AMAP_KEY` 环境变量读取，默认使用你提供的 `e6b2ed8c6343ef98e8dbae4e81ec2cac`。
- 小程序已增加管理员后台：在「我的」里切换到主管/运营账号，可进入管理员后台给员工开账号、设置角色和区域。
- 员工看板已按当前账号过滤：销售只看自己的客户、跟进和问题分析；主管/运营/区域经理看团队数据。
- 名单/线索/商机/成交已支持一级一级推进，客户可单个添加、粘贴导入，也可上传 CSV/TXT 名单文件导入。
- 小智 AI 话术模块已接后端 `/api/ai/script`，后端会从知识库检索并调用 DeepSeek 生成跟进策略。真实调用需要配置环境变量 `DEEPSEEK_API_KEY`；未配置时会自动使用知识库本地策略。
- 知识库支持小程序端持续新增，新增后 Web/小程序共享。
- App 自动录音需要 Android/iOS 原生能力或企业级通话 SDK；普通网页无法自动录音电话。
- 高德地图需要申请 Web/移动端 Key 后替换当前模拟地图。
- 生产版建议增加后端 API、数据库、对象存储、ASR 语音转写、RAG 知识库和权限审计。
